ABOUT HTML

HTML

HTML is the standard markup language for creating Web pages. HTML stands for Hyper Text Markup Language. It describes the structure of a Web page. It consists of a series of elements. It elements tell the browser how to display the content. It elements are represented by tags.

HTML TAGS

HTML tags normally come in pairs. The first tag in a pair is the start tag, the second tag is the end tag The end tag is written like the start tag, but with a forward slash inserted before the tag name. The start tag is also called the opening tag, and the end tag the closing tag.
